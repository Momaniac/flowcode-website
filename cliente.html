FLUJO DE TRABAJO

Orquestación en 3 Pasos

01

Inicialización

El desarrollador descarga el perfil del proyecto asignado por el Tech Lead. Las reglas de linter y arquitectura se configuran solas.

02

Contexto Local

Se selecciona el directorio de trabajo. Flow Code crea un índice vectorial local para entender tu código sin subirlo.

03

Generación

Solicitas cambios complejos ("Refactoriza el módulo de pagos"). El agente ejecuta y tú validas el Diff.
